# Contributing

Thanks for your interest in Karma! Contributions, issue reports, and suggestions are welcome.

How to get started

- Clone the repo and install locally:

  git clone https://github.com/sinhara666/karma.git
  cd karma
  npm link

- Run the offline CLI:

  karma-offline

- Try the example linter:

  # from the project root
  node ./karma-offline-cli.js
  # at the prompt
  lint example.js

Guidelines for contributions

- Open an issue first to discuss larger changes.
- Fork the repo, create a feature branch, and open a pull request.
- Keep changes focused and add tests/examples when possible.
- Do not commit secrets or API keys.

Code style

- Use clear, small commits with descriptive messages.
- JavaScript files use modern ES module syntax (type: module in package.json).
